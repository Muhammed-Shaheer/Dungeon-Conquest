# Dungeon Conquest

A dungeon crawler game built with Python and Pygame.

## Current Features

- Class Selection System
- 2 Playable Classes
  - Tank
  - Swordsman
- 4 Slime Enemy Types
  - Fire Slime
  - Water Slime
  - Mud Slime
  - Ice Slime
- Boss Battle
  - King Slime
- Trap System
- Player Movement
- Combat System
- Health System
- Enemy AI
- Game Over Screen
- Floor Clear Screen

## Requirements

- Python 3.x
- Pygame

## Installation

```bash
pip install pygame
```

## Run

```bash
python main.py
```
